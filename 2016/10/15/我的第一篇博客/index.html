如何用hexo创建自己的博客

发表于 2016-10-15

学习hexo创建个人博客并放到github上

hexo介绍

hexo是基于nodejs开发的一个软件,也成为工具,所以我们必须要有一个nodejs环境,首先打开git Bash Here输入node -v 确定你的电脑是否安装nodejs,然后验证npm -v ,没有的话先安装nodejs.

创建步骤

  • 在任意一个盘下打开git Bash Here,执行命令hexo,hexo就是用来做博客的,他路径不能随便乱写.选好路径后(比如在E盘下)创建一个文件夹,怎么创建呢?不能直接新建文件夹,要用git Bash Here(以后简称git命令行) 输入hexo init blog(文件命名自己随便写)没网的情况下也可以创建,自身会克隆.在git命令行输入ls 看下有没有这个文件夹.创建好后执行在git命令行输入 cd blog/ 进入blog文件内,打开git命令行.
  • 在git命令行输入hexo server或hexo s 启动本地一个服务器，用来预览我们的博客 在网页上输入localhost:4000,就进入hexo默认的博客.
  • 如何创建属于自己的博客呢,在git命令行输入 hexo new 我的第一篇博客,在blog文件里的source里的_post就有你的 我的第一篇博客.md文件,这样就用你下载好的markdown书写你的博客了.
  • 如果你觉得hexo提供的主题界面太丑了 就需要自己下载你喜欢的主题了,或者自己写个页面,hexo官网有很多主题,下载地址https://hexo.io/themes/,下载好的主题放到themes里,然后打开blog里的config文件,用sublim打开 ,把里面的theme改成你下载好的主题文件名字,修改好保存,在git命令行输入ctrl+c 退出blog目录下,输入hexo s 从新启动 ,同时主题里也有好多样式共你选择,打开下载的主题文件,用sublim或记事本打开里面是config文件,找到schemes文件修改样式即可.
  • 如何修改成简体中文 打开下载好是主题文件知道language文件夹,找到zh-Hans.yml(中文输入),复制文件名,打开站点(blog文件下)的config ,找到language 粘贴下(加双引号).
  • 如何想要快速的发送到github上呢,需要在blog下打开git命令行输入 npm install hexo-deployer-git
  • 打开站点下是config文件 找到并输入 type:git repo:你的github创建的ssh地址 branch:master
  • 输入hexo g 意思是讲md转化成真实的html文件,再输入hexo d 意思是部署到github上
